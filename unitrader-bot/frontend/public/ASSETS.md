# Public Assets

Drop the following files here:

| File | Used for |
|---|---|
| `logo-ring.png` | Spinning loader, navbar logo, browser tab favicon, apple touch icon |

Recommended sizes:
- `logo-ring.png` — 512×512 px, transparent background, PNG or WebP
- Copy the same file as `favicon.ico` or use a 32×32 version for the `.ico`

Once `logo-ring.png` is present, the loader spins, the navbar shows the ring,
and the browser tab icon updates — no code changes needed.

# Public Assets

Drop the following files here:

| File | Used for |
|---|---|
| `logo-galaxy.png` | Spinning loader, navbar logo, browser tab favicon, apple touch icon |

Recommended sizes:
- `logo-galaxy.png` — 512×512 px or larger, square crop preferred
- Copy the same file as `favicon.ico` or use a 32×32 version for the `.ico`

Once `logo-galaxy.png` is present, the loader spins, the navbar shows the galaxy,
and the browser tab icon updates — no code changes needed.
